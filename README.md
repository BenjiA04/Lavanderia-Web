# LavaYa

Página web moderna para una lavandería desarrollada con HTML, CSS y JavaScript.

## Características
- Diseño responsive
- Formulario de solicitud
- Sección de servicios
- Navbar interactiva
- Diseño moderno y limpio

## Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript
